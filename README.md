# calculator-reactJS

## 功能说明
输入数字，选择 `+` , `- `, `*` , `/`，点击 `=`按钮得到结果

##涵盖技术
- git & github 
- react & react-dom
- redux & react-redux & redux-logger 
- webpack & webpack-dev-server
- babel & ES6
- eslint


##how to use

```
git clone https://github.com/vickyzhu2017/calculator-reactJS.git
cd calculator-reactJS & npm install
npm run start 

```
